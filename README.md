## Introduce

###### 功能简介

本项目实现了一个简单的后台管理系统，可以作为ssm+dubbo项目学习的脚手架，主要包含以下功能： <br /> 
- 管理员的注册功能，登录功能，删除功能。 <br /> 
- 文章的增删改查功能，图片的增删改查功能。 <br /> 
- 图片上传功能。 <br /> 
- 多文本编辑器UEditor整合。 <br /> 

###### Framework

- Spring <br /> 
- SpringMVC <br /> 
- MyBatis <br /> 
- dubbo 2.6.5 <br />
